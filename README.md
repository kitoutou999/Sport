# Sport

A minimal browser-based daily workout tracker. Displays a list of exercises based on the current day of the week, with the ability to mark them as done and reset.

## Features

- Day-based exercise list (auto-generated from day of week)
- Mark exercises as completed
- Reset the day's progress

## Usage

Open `index.html` in your browser. No installation required.

## Tech Stack

- Vanilla JavaScript
- HTML/CSS
